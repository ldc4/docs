# 手册

手册的意义

目标如果是：认知变现

认知：了解、认识客观事物而获得知识。

认知是很重要的东西，它可以拓宽视野，也可以为你带来其他收益

它是由每个人在生活过程中一点一点产生出来的，

很大部分的认知都是需要通过其他途径来吸收，

但随着长时间不使用，记忆会遗忘，下次用到又得到处寻找，

所以需要有个地方沉淀，让其能够帮助快速找回记忆
